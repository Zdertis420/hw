# Описание
Данный проект преобразует файл на выдуманном языке конфигурации в toml и выводит в консоль

# Функции и настройки
Программа принимает только исходный файл в качечтве аргумента командной строки

# Сборка и запуск
## Зависимости проекта
- Python 3.13.7 или новее
- pip 25.3 или новее
- lark 1.3.1 или новее
- toml 0.10.2 или новее

## Сборка
Установите все необходимые зависимости таким образом, который является правильным для вашей ОС.
Здесь будет демонстрация сборки под Arch Linux

### Установка и сборка
Установите Python и pip
```bash
sudo pacman -S python pip
```

Склонируйте репозиторий и перейдите в его директорию
```bash
git clone https://github.com/Zdertis420/hw
cd hw
```

Создайте виртуальное окружение Python и утсновите необходимые зависимости
```bash
python -m venv venv
source venv/bin/activate
pip install reqs.txt
```

Проект собран и готов к использования

## Запуск
Запуск осуществляется простым вызовом интерпретатора
```bash
python src/main `путь до файла конфигурации`
```
